# Realtime Web Chat Application

A lightweight, serverless chat application built with vanilla **HTML5**, **CSS3**, and **JavaScript (ES6+)**, utilizing **Firebase** for the backend infrastructure.


## 🚀 Key Features

* **Realtime Messaging:** Powered by Firebase Firestore snapshot listeners for instant message delivery.
* **User Authentication:** Secure email/password or Google Sign-In via Firebase Auth.
* **Responsive UI:** Clean, modern, and mobile-friendly interface built with pure CSS.
* **Serverless Architecture:** No backend server configuration required.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
* **Backend-as-a-Service (BaaS):** Firebase (Firestore, Authentication)
